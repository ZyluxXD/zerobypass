# Zerobypass

Proof of concept tool to bypass document replay technology (such as GPT Zero). It uses Playwright automation to type
like a human.
___

## Requirements

- Python 3.11 or newer
- Display and clipboard access.

___

## Quick start

1) Install using either method:
2) 
3) `pipx install zerobypass` (recommended)
4) `pip install zerobypass`
5) `git clone https://github.com/Magic-Goose-Software-Co/zerobypass.git`
6) 
7) (Optional) Preinstall the browser bundle: `python -m playwright install --with-deps chromium`
8) Run: `zerobypass` (or `python -m src` from the repo).

___

## Notes

- This is still a **Work In Progress** and is **not** finished.
- The Playwright user data directory lives at `~/.zerobypass`; delete it to reset the profile.
